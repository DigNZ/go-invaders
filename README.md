# Go Invaders

8080 Emulator for playing Space Invaders written in Go.

Following details from http://emulator101.com/

Uses Raylib via github.com/gen2brain/raylib-go/raylib

| Key    |   Action      |
| ------ | ------------- |
|  C     | Insert Coin   |
|  Space | Shoot         |
| Left   | Left          |
| Right  | Right         |
| 1      | Player1 Start |
